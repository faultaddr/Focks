#### Focks

---

Focks 是一款设计为Forest替代品的免费软件，主要用于个人时间的管理。支持设定时间后再玩手机一秒变猪，当前仍在开发阶段。开发者为Panyunyi（潘云逸），
开发者对所有使用具有最终解释权。遵循GPL协议，严格不准商用。
